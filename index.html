A continuación, se encuentran definidas, algunas de las etiquetas de HTML, ya que son bastantes.

Descripción de las etiquetas de HTML
Etiqueta Descripción
<!– – Comentario – –> Es la etiqueta que es utilizada para realizar comentarios en HTML
<!DOCTYPE> Declaración del tipo de documento, permite al navegador entender qué versión de HTML se está utilizando.
<head> Define información hacerca del documento, es la primera de las dos partes en que se estructura un documento HTML.
<body> Define el cuerpo del documento, representa el contenido de un documento HTML. Sólo puede haber un elemento <body> en un documento.
<header> Define la sección de encabezado del docuemnto, representa un grupo de ayudas introductorias o de navegación. Puede contener algunos elementos de encabezado.
<div> Define una sección en un documento, definir un bloque de contenido o sección de la página, que se pueden aplicar diferentes estilos e incluso para realizar operaciones sobre ese bloque específico.
<form> Define un formulario html, permite al usuario introducir datos los cuales son enviados a un servidor para ser procesados.
<input> Define un control de entrada de texto, se usa para crear controles interactivos para formularios basados en la web, que reciban datos del usuario.
<label> Define el rótulo para un elemento <input>, representa una etiqueta para un elemento en una interfaz de usuario.
<link> Define la relación entre un documento y un recurso externo (generalmente con hojas de estilo), es decir, especifica la relación entre el documento actual y un recurso externo.
<footer> Define el pie de página de un documento, generalmente se incluyen links de navegación, enlaces de interés, copyright o botones a las redes sociales del site
<nav> Define una sección que solamente contiene enlaces de navegación

    • A continuación, se encuentran definidas, algunas de las etiquetas de CSS, que son las encargadas de dar estilo a las paginas.

    Descripción de etiquetas CSS
    Etiqueta Descripción
    <font-family> Define una lista de fuentes o familias de fuentes, con un orden de prioridad, para utilizar en un elemento seleccionado.
    <color> Define el color del texto.
    <font-size> Permite establecer el tamaño del texto de forma relativa mediante las palabras clave larger y smaller.
    <text-align> Se encarga de la alineación del texto, no sólo controla la alineación del texto. El valor inicial de la propiedad text-align depende a su vez del valor de la propiedad direction.
    <border> Es una de las "propiedades shorthand" que define CSS y que se utilizan para establecer de forma abreviada el valor de una o más propiedades individuales
    <margin> Establece el margen para los cuatro lados. Es una abreviación para evitar tener que establecer cada lado por separado con las otras propiedades de margen.
    <padding> Establece el espacio de relleno requerido por todos los lados de un elemento. El área de padding es el espacio entre el contenido del elemento y su borde ( border ) no se permiten valores negativos.
    <background-color> Es un propiedad de CSS que define el color de fondo de un elemento, puede ser el valor de un color o la palabra
    <background-image> Establece cualquier imagen como fondo de un elemento.
    <list-style> Permite establecer en una sola propiedad los valores de las propiedades
    <border-collapse> Permite elegir el modo de presentación de los bordes de las celdas y de la tabla. Los dos valores permitidos son separate (separado) y collapse (colapsado).
    <display> Permite especificar el tipo de caja que se aplica en un elemento. Los diferentes tipos de cajas corresponden a diferentes tipos de contenido (texto, listas, tablas).
    <position> Establece el esquema de posicionamiento absoluto de un elemento. Existen cuatro tipos en el esquema de posicionamiento absoluto; (static, relative, fixed y absolute).
    <list-style-position> Establece la posición del marcador con respecto al resto del elemento de la lista: dentro (inside) o fuera (outside). los navegadores aplican por omisión el valor outside.
